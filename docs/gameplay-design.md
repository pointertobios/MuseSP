# MuseSP 玩法设计文档

## 概述

MuseSP 是一款基于球面投射的音游。所有判定发生在半径 r=20 的球面上。

## 核心概念

### 判定面

- 以原点为中心、半径 r=20 的球面
- 所有判定在此球面上进行

### 判定区（九宫格，3×3）

- 球面上的 3×3 正方形网格，共 **9 个音符轨道**
- 九宫格**跟随鼠标移动**：鼠标在球面上的投射点即为九宫格中心
- 鼠标未命中球面时九宫格隐藏
- 轨道编号（俯视，面向 +Z 方向）：

```
  [0] [1] [2]    ← 上排
  [3] [4] [5]    ← 中排
  [6] [7] [8]    ← 下排
```

- 每个轨道的角尺寸：12° × 12°，整个九宫格约 36° × 36°

### 键盘触发区（四宫格，2×2）【待实现】

- 球面上的 2×2 正方形网格，**位置固定**
- 映射 4 个键盘按键（推荐 D / F / J / K）
- 当九宫格的某个轨道进入触发区范围内时，对应按键可触发击打

```
  [D] [F]
  [J] [K]
```

- 触发区角尺寸：与九宫格每个单元格相同

### 触发逻辑【待实现】

```
九宫格轨道[i] 与 触发区格子[j] 重叠
  → 按下 key[j] 击打轨道[i] 上最接近球面的音符
```

### 音符系统【待实现】

#### 音符运动

- 音符从**原点 (0,0,0)** 向**判定面（球面 r=20）** 沿径向匀速发射
- 音符到达球面时即为"判定时机"
- 音符速度可配（`gameplay.notespeed`）

#### 音符类型

| 类型         | 说明                                          |
| ------------ | --------------------------------------------- |
| 单击（Tap）  | 到达判定面时按下对应按键                      |
| 长按（Hold） | 起始格到结束格，可横跨 2~3 个轨道，需按住按键 |

#### 轨道切换

- 长按音符在飞行过程中，玩家**移动鼠标**可改变音符的目标轨道
- 音符始终沿径向飞行，目标轨道由当前位置的九宫格决定

## UI / 交互

### 光标

- 进入 GameplayPage 时自动隐藏系统光标
- 始终显示绿色十字指针（24×24 px）代替光标
- 离开页面时恢复系统光标

### 鼠标 → 球面拾取

1. 屏幕坐标 (sx, sy) → NDC (-1..1)
2. 逆 view_proj 矩阵 → 世界空间射线
3. 射线-球面求交 → 交点 Cartesian 坐标
4. Cartesian → 球坐标 (θ, φ)

## 摄像机系统

### 配置文件

摄像机初始参数从 `config.example.toml` 的 `[gameplay.camera]` 加载：

```toml
[gameplay.camera]
eye = [35.0, 25.0, 35.0]          # 摄像机位置
direction = [-0.631, -0.451, -0.631]  # 朝向（单位向量）
up = [0.0, 1.0, 0.0]              # 上方向
fov_degrees = 60.0                 # 视场角
near = 0.1
far = 200.0
```

### 摄像机调整模式

- 开关：`debug.gameplay.camera_adjustable = true`
- 快捷键：**Ctrl+1** 切换 Playing ↔ Adjusting
- Playing 模式：正常游戏，九宫格跟随鼠标
- Adjusting 模式：九宫格冻结，可调整摄像机

| 操作         | 说明                 |
| ------------ | -------------------- |
| W/A/S/D      | 前/左/后/右平移      |
| Space        | 上升                 |
| LAlt         | 下降                 |
| 鼠标移动     | 旋转视角（第一人称） |
| 滚轮         | 调整 FOV（10°–120°） |
| Esc / Ctrl+1 | 切回 Playing 模式    |

- 退出页面时自动保存当前摄像机参数到配置文件（仅当有修改时）

## 架构

```
musesp/src/gameplay/
├── renderer3d.rs    # 渲染管线 + 几何生成 + 鼠标拾取 + 摄像机状态
└── gameplay.rs      # 模块入口
```

核心类型：

| 类型                     | 位置               | 说明                                     |
| ------------------------ | ------------------ | ---------------------------------------- |
| `GameplayState`          | `renderer3d.rs`    | 全局状态（鼠标、球面拾取、摄像机、模式） |
| `CameraState`            | `renderer3d.rs`    | 摄像机（eye、direction、up、fov）        |
| `CameraMode`             | `renderer3d.rs`    | Playing / Adjusting                      |
| `GameplayRenderPipeline` | `renderer3d.rs`    | 自定义渲染管线（两 pass 自适应细分）     |
| `AsyncSnapshotProducer`  | `renderer3d.rs`    | 异步几何预计算（tokio 后台任务）         |
| `GameplayPage`           | `gameplay_page.rs` | 页面入口（生命周期、事件分发、输入处理） |
